Áp dụng trong hệ điều hành window

Hướng dẫn biên dịch và chạy chương trình

1. Tải xuống và Cài đặt môi trường java theo đường link sau (nếu chưa có):
	https://java.com/en/download/windows-64bit.jsp

2. Giải nén file nén chứa mã nguồn chương trình

3. Nhấn đúp vào file: CompileAndRun.bat để biên dịch và chạy chương trình một cách tự động


Hướng dẫn chơi game:

- Sau khi chạy chương trình có 3 chế độ chơi:
    + Single Play: chơi theo chế độ cổ điển (chơi một mình)
    + Play with Computer: chơi đối kháng với máy
    + Auto Play: cho máy tự chơi
- Để điều khiển con rắn dùng phím A, S, W, D hoặc LEFT, DOWN, UP, RIGHT tương ứng với hướng di chuyển trái, xuống, lên, phải của con rắn.
